# 🎯 DigiTools - Digital Tools Buying Website

## 📌 Description
DigiTools is a modern digital tools marketplace where users can explore premium digital products and add them to cart. It provides a smooth user experience with product browsing, cart management, and checkout functionality.

---

## 🚀 Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify

---

## ✨ Features
1. 🛒 Add to cart with real-time update and toast notification  
2. ❌ Remove products from cart and clear all with checkout  
3. 🔄 Toggle between Products and Cart with smooth UI
